## Install

To install dependencies:

```bash
nvm install
nvm use
yarn install
```

## Usage

To build contracts including all sanity checks:

```bash
yarn build-full
```

To build contracts quickly during development:

```bash
yarn build-quick
```

To clean repository:

```bash
yarn clean
```

To run tests:

```bash
yarn test-coverage
yarn test-gas
```

For more, see `package.json`
